# Papaya – Thai Alphabet Lernapp

Eine interaktive Web-App zum Lernen des thailändischen Alphabets — kostenlos, ohne Account, direkt im Browser.

## Was kann die App?

- **25 Lektionen** mit Konsonanten, Vokalen, Tonzeichen und Zahlen
- **Lernmodus** – Karteikarten mit Aussprache-Tipps, Beispielwörtern und Tonklassen
- **Quizmodus** – Multiple-Choice-Fragen, 80% zum Bestehen nötig
- **Fortschritt** wird lokal im Browser gespeichert (kein Account nötig)
